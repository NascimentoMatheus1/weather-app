# Weather app
- Um weather app (aplicativo de clima) é um aplicativo que fornece informações meteorológicas em tempo real e previsões do tempo para localizações específicas.
- As informações estão em inglês e no sistema imperial.
- [Visualização (Live Preview)](https://nascimentomatheus1.github.io/weather-app/)
 ## Como utilizar:
- Clique no Live Preview
- Digite: cidade, estado ou cidade, país
- Aperte Enter ou o botão Search para visualizar os dados
## Habilidades Praticadas:
- Trabalho com APIs externas
- Programação assíncrona
- Uso de JavaScript assíncrono (promises ou async/await)
- Manipulação de JSON
- Interface de usuário dinâmica
- Gestão de estado da aplicação
